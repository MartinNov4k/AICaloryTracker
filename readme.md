# Jídlovize 🍏

Aplikace pro sledování denního příjmu kalorií a makroživin. Využívá OpenAI API k automatickému rozpoznávání jídel z fotografií a textu pro výpočet nutričních hodnot. Uživatelské rozhraní je postavené na frameworku Streamlit.



## 🚀 Live Demo
Aplikace je nasazená online. Pro otestování funkcí můžete využít sdílený testovací účet:
* **Odkaz:** [Jídlovize na Streamlitu](https://calorytracker.streamlit.app/)
* **Přihlašovací jméno:** `admin`
* **Heslo:** `123`
*(Poznámka: Jedná se o veřejný ukázkový účet.)*

## ✨ Hlavní funkce
* **AI detekce:** Rozpoznání složení jídla a odhad porce (v gramech) z fotografie nebo textového popisu.
* **Nutriční výpočty:** Automatický výpočet makroživin (kalorie, bílkoviny, sacharidy, tuky, cukry).
* **Data vizualizace:** Denní přehled a interaktivní vizualizace plnění cílů pomocí knihovny Plotly.
* **Backend:** Ukládání a načítání historie jídel přes externí REST API.
