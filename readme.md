# Jídlovize

Aplikace pro sledování denního příjmu kalorií a makroživin. Využívá OpenAI API k automatickému rozpoznávání jídel z fotografií a textu pro výpočet nutričních hodnot. Uživatelské rozhraní je postavené na frameworku Streamlit.

## Funkce
* Rozpoznání složení jídla a odhad porce (v gramech) z fotografie nebo textového popisu.
* Automatický výpočet maker (kalorie, bílkoviny, sacharidy, tuky, cukry).
* Denní přehled a interaktivní vizualizace plnění cílů (Plotly).
* Ukládání a načítání historie jídel přes externí REST API.
